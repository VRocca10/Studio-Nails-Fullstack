# Studio Nails

Projeto atualizado

## Tecnologias

- React
- Vite
- Supabase
- React Router

## Funcionalidades adicionadas

- Banco de dados PostgreSQL
- Login de usuários
- Login administrativo
- Estrutura para agendamentos
- Upload de imagens
- CRUD de serviços
- CRUD de galeria
- Formulário de contato persistente

## Instalação

```bash
npm install
```

## Configuração

Copie:

```bash
.env.example
```

para:

```bash
.env
```

e preencha os dados do Supabase.

## Executar

```bash
npm run dev
```

## Banco de dados

Execute:

```sql
supabase-schema.sql
```

no SQL Editor do Supabase.
